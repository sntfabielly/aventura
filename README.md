## 📋 Descrição

Este projeto é um **jogo de aventura interativa** onde o jogador faz escolhas que determinam o rumo da história. Possui:

- 🎮 **Tela de capa** com design temático rosa e botão "Iniciar"
- 📖 **Sistema de história** com múltiplas escolhas e caminhos
- 💫 **Feedback visual** a cada decisão
- 📊 **Progresso** e contador de escolhas
- 🎨 **Animações** e efeitos visuais (confetes, transições)
- 📱 **Responsivo** para todos os dispositivos
